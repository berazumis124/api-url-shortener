URL shortener Microservice

Usage example:

http://localhost:4998/add/https://google.com

https://timestamp-api-mr.herokuapp.com/1493326800

Output example:

{"full_url":"https://google.com","short_url":"http://localhost:4998/31"}

Using short url:

Point browser to http://localhost:4998/31 and it will be redirected to original url.
